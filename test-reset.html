Testing Reset Functionality

Open the browser console and check for any errors when clicking the "Play Again" button.

The application should be running at http://localhost:3000

Expected Behavior:

  • When a game ends (win or draw), a "Play Again" button should appear
  • Clicking "Play Again" should reset the board and start a new game
  • The chat should be cleared when the game resets
  • Both players should be able to make moves in the new game

Debugging Steps:

  1. Open two browser windows/tabs to http://localhost:3000
  2. Play a game until someone wins or it's a draw
  3. Click "Play Again" button
  4. Check browser console for any error messages
  5. Verify that the board resets and both players can play again
